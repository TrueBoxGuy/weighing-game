# 12 Coins Puzzle

A web-based AI visualization of the classic 12 coins weighing puzzle, built with Rust and Dioxus. This is not my code (it's the code of an AI directed by me). It was built with Antigravity and Gemini 3. 

## Built With
- **Rust**: Core logic.
- **Dioxus**: UI framework for web assembly.
- **Tailwind CSS**: Styling.

## Development

Local:
```bash
trunk serve
```
